# Practice Assessment - AZ-400

The single most exam-aligned free resource Microsoft publishes.

---

## Official Microsoft practice assessment

[**AZ-400 Practice Assessment**](https://learn.microsoft.com/credentials/certifications/devops-engineer/)

- **Free**, unlimited retakes.
- Same question style and difficulty as the live exam.
- Each question shows the correct answer + a link to the relevant Microsoft Learn module.
- Score reports break down by **functional group** so you know which domain to revisit.

> **Workflow:** Take it cold once -> review **every** answer (including the ones you got right - Microsoft sometimes accepts 2 of 4 valid answers). Re-study weak areas. Take it again. Aim for >=85% twice in a row before booking the live exam.

---

## How the live exam differs

- **40-60 questions, 100 minutes** (varies - Microsoft no longer publishes exact counts).
- Includes **case studies** (~5 questions on a shared scenario; you cannot revisit after submitting).
- May include **labs** (live Azure portal tasks) - read the entire task list before starting.
- **No back-tracking** within case studies; you can review unmarked questions in the main section.
- Passing score: **700/1000** (scaled, not a flat percentage).

---

## Question patterns to expect

| Pattern | Trick to watch for |
|---|---|
| "Drag the steps into the correct order" | Order matters - read all options first; some look right but are reversed |
| "Select 3" | Wrong number = 0 points, even if 2 of 3 are correct |
| "Yes / No / Yes" series on same scenario | Each is independent - don't anchor on the first answer |
| "Most cost-effective" | Always picks the **cheapest valid** option, not the best |
| "Minimum admin overhead" | Picks **PaaS / managed / fully integrated** option |
| "Comply with regulation X" | Look for the option that includes **Azure Policy + audit log + retention** |
| "Minimum permissions" | Pick the **smallest scope + least privileged** built-in role |

## Other practice resources (use sparingly)

- **MeasureUp** - paid, generally considered the closest 3rd-party simulator. Buy the **30-day** subscription only.
- **Whizlabs / ExamTopics** - heavy bias toward outdated content. Cross-check every answer against [Microsoft Learn](https://learn.microsoft.com).
- **AI Copilot quizzes** - see the [AI Copilot Quiz](17-copilot-quiz.md) page in this guide. Free, infinite, conceptually accurate.

> **Warning:** Using "exam dumps" (verbatim leaked questions) violates Microsoft's certification policy and gets your account permanently banned. Stick to practice assessments and conceptual drills.
